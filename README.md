# Onega Mission

## Проект для Арктического хакатона

`Onega Mission` создан в рамках **Арктического хакатона** как социально-экологическая игровая инициатива.
Главная идея проекта - наглядно показать, к каким последствиям приводит мусор в северной природе.

## Цель проекта

- Показать ужасы загрязнения окружающей среды через интерактивный игровой опыт.
- Наглядно продемонстрировать проблему мусора в Арктике.
- Обратить внимание на хрупкость экосистемы региона Белого моря.
- Сформировать у игрока чувство личной ответственности за состояние природы.

## Сеттинг

События происходят на острове в акватории **Белого моря**.
Игрок видит контраст между красотой северной природы и следами человеческого загрязнения.

В мире игры встречаются животные Арктики:
- тюлени;
- чайки.

## Роль игрока

Игрок выступает в роли **стажера-эколога**.
Его задача - очистить остров от мусора, помочь восстановить природную среду и пройти путь от наблюдателя к активному защитнику природы.

## Идея воздействия

Проект делает акцент не только на игровом процессе, но и на просветительской функции:
через действия игрока демонстрируется, что даже локальная уборка может стать важным шагом к сохранению Арктики.

## Структура проекта и билд

- Unity-проект находится в папке `OnegaMission`.
- ПК-билд находится в папке `ASD`.
- Исполняемый файл игры (`.exe`) лежит внутри `ASD`.

## Управление

- `W`, `A`, `S`, `D` - движение.
- `Мышь` - поворот камеры, взаимодействие с сортировочными корзинами, UI и предметами.
- `1`, `2`, `3` - выбор инструментов: планшет, ножницы, камера.
- `E` - взаимодействие.
- `I` - активация планшета.
- `P` - вход в фоторежим.
- `E` (в фоторежиме) - сделать фото.
- `Q` - выход из режима сортировки и выход из фоторежима.
